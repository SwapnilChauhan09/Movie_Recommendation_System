Movies :

budget : Movie's budget.
genre : The genre, in which movie belongs to(e.g. - Action, Comedy ,Thriller etc).
homepage : A link to the homepage of the movie.
id : A unique identifier for each movie.
keywords : The keywords or tags related to the movie.
original_language : The language in which the movie was made.
original_title : The title of the movie before translation or adaptation.
overview : A brief description of the movie.
popularity : A numeric quantity specifying the movie popularity.
production_companies : The production house of the movie.
production_countries : The country in which it was produced.
release_date : The date on which it was released.
revenue : The revenue generated by the movie(worldwide).
runtime : The running time of the movie in minutes(screen time).
status : "Released" or "Rumored".
tagline - Movie's tagline.
title : Title of the movie.
vote_average : average ratings the movie recieved.
vote_count : the count of votes recieved.


Credits : 

movie_id : A unique identifier for each movie.
cast : The name of lead and supporting actors.
crew : The name of Director, Editor, Composer, Writer etc.
